Изучение HTML

Текстовые теги

Для красивого оформления текста разработчики используют в HTML специальные теги, которые выделят текст на фоне остальных, например, текст можно делать полужирным, курсивным, перечёркнутым , подчёркнутым и так далее. При этом ничто не мешает совмещать несколько стилей текста.

С помощью CSS стили текста можно изменять, например, назначить другой шрифт, размер или отступы

Подробнее про HTML-теги можно узнать воспользовавшись поисковиком

Формула

HTML также позволяет встраивать формулы. Например, ниже приведена формула теоремы Пифагора
